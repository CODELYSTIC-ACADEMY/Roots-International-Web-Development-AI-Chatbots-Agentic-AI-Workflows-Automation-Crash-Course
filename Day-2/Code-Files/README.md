## Important Teaching Note

Do not ask students to paste the full final code at once.

Teach it block by block:

1. Boilerplate
2. First heading and paragraph
3. About section
4. Skills list
5. Link and button
6. Semantic tags
7. Header and navigation
8. Hero section
9. Projects section
10. Contact form
11. Footer

After every few lines of code, ask:

> What changed on the website?
