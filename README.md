## 🗣️ Key Concepts Introduced on Day 01

### What is Web Development?

Web development is the process of creating and improving websites and web applications.

It includes:

- Planning
- Coding
- Designing
- Testing
- Publishing

---

### HTML, CSS, and JavaScript

| Technology | Purpose | Simple Meaning |
|---|---|---|
| HTML | Structure | Creates headings, text, images, buttons, and sections |
| CSS | Design | Adds colors, layout, spacing, fonts, and style |
| JavaScript | Interactivity | Adds actions like button clicks, dark mode, and form messages |

---

### GitHub

GitHub is used to:

- Save code online
- Share projects
- Track progress
- Build a developer profile
- Collaborate with others

---

### Vercel / GitHub Pages

Vercel and GitHub Pages are used to:

- Publish websites online
- Create live project links
- Share work with others
- Showcase projects professionally

---

### AI Chatbot

An AI chatbot is a conversational system that can answer questions.

Examples:

- Course FAQ chatbot
- School information chatbot
- Study helper chatbot
- Website support chatbot

---

### AI Assistant

An AI assistant can help complete tasks, such as:

- Creating study plans
- Generating quizzes
- Explaining concepts
- Helping with website content
- Organizing information

---

### Agentic AI Workflow

An agentic AI workflow is when AI follows a goal and completes multiple steps using tools.

Example:

```text
Student submits a form
↓
AI reads the input
↓
AI decides what help is needed
↓
AI creates a useful response
↓
The result is saved or sent
```

## ⚠️ Important Course Note

This is a one-month crash course.

Students will not become professional developers in one month, and that is completely okay.

The goal is to:

- Build strong foundations
- Learn practical tools
- Create real projects
- Understand modern AI skills
- Start the learning journey confidently

---

## ❌ Why We Are Not Starting With Advanced Frameworks

Advanced tools like React, Next.js, APIs, databases, and full-stack systems are powerful, but they are too technical for the beginning stage.

Before learning advanced frameworks, students need strong foundations in:

- HTML
- CSS
- JavaScript
- Website structure
- Responsive design
- Deployment
- Project presentation

> Strong fundamentals make advanced development easier later.
